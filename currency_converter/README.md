# Currency Converter (Live CNB Data)

**Description:** A desktop GUI application that converts CZK to EUR using real-time exchange rates from the Czech National Bank.
**Core Technologies:** Python, Tkinter, Requests API.

**Key Features:**
* Fetches current daily rates via `requests` directly from CNB.
* Data parsing and cleaning (handling decimal commas vs. dots).
* Intuitive GUI with error handling for invalid user inputs.
